# Employee Self-Service Portal

## Project Overview

The Employee Self-Service Portal is an internal web-based application designed to manage and streamline employee-related requests within an organization. It provides a centralized platform where employees can submit requests and managers can review, approve, or reject them.

The system supports different types of requests, including Leave Request, Vehicle Request, Recommendation Letter Request, and Equipment Request. It is implemented using React for the frontend, Laravel for the backend, and SQLite as the database, following the Model-View-Controller (MVC) architectural pattern.
